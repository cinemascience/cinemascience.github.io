Cinema Examples

Cinema Spec A

A simple sphere with Spec A: a single theta=0 value with 12 phi values. This interactive visualization uses the Cinema desktop viewer. Zooming can be done with the mouse scroll wheel. Rotation can be done via mouse or UI controls.

Cinema Spec A

A Spec A database of 7440 images from a Nyx cosmology simulation of dark matter density. There are 31 time steps and multiple theta/phi values. Additionally, there are five different isosurfaces available.

Cinema Spec C

A Spec C database of a cosmology dark matter halo simulations. The halo cores and the volume visualization are saved separately and can be turned on and off separately enabling different views of the information available in the data. The data is saved as a function of time, theta, and phi.

Cinema:Explorer - Spec D

The parallel coordinates view is used to explore the Nyx simulation. The parallel coordinates view can be used to select images with similar properties. The scatterplot can be used to explore correlations between variables. The variables in the database include time, phi, theta, entropy, sample number, and data abstracts such as Hu moments and Canny edge value.
